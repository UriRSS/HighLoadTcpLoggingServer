High-Load TCP Logging Server
Высокопроизводительный TCP-сервер для приёма и сохранения логов. Поддерживает тысячи одновременных подключений.

- Асинхронная обработка на .NET 8
- Ограниченный канал (Channel) для буферизации сообщений
- UTF-8 валидация входящих данных
- Автоматическое отключение неактивных клиентов (таймаут 2 мин)
- Групповая запись в файл (flush каждые 100 сообщений)
